GoalFlow

personīgais ceļvedis mērķiem un iedvesmai

Sekojiet savam progresam, atklājiet motivējošas idejas un vizuāli sakārtojiet savus mērķus. Viss vienuviet — lai sasniegtu vairāk un justos iedvesmots katru dienu. Vietne apvieno praktisku attīstību ar estētisku baudījumu, padarot progresu redzamu un motivējošu.

Galvenas funkcijas

feature 1

Mērķi un progress: seko saviem mērķiem, redzi progresu un vizuāli atzīmē sasniegumus.

feature 2

Iedvesmas plūsma: personalizēta lapa ar skaistām bildēm un citātiem, kas atbilst tavām mērķu tēmām.

feature 3

Mindmap: vizuāli veido mērķu karti un sasaisti idejas, lai labāk redzētu savu ceļu.

Stack

Frontend: React + TailwindCSS — interaktīvs un vizuāli skaists interfeiss. Datu bāze: Postgres caur Supabase — visi mērķi, progresi un iedvesmas resursi glabājas droši mākoņā. Backend: Supabase API — gatavs risinājums datu pārvaldībai un autorizācijai, savs serveris nav nepieciešams. Mindmap: jsMind (vai React Flow) — vizuālās mērķu kartes veidošanai. Iedvesma: Unsplash API (bildes). Progresu vizualizācija: Chart.js — grafiki un izpildes indikatoru vizualizācija.
