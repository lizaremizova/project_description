Galvenas funkcijas

Stack

Frontend: React + TailwindCSS — interaktīvs un vizuāli skaists interfeiss. Datu bāze: Postgres caur Supabase — visi mērķi, progresi un iedvesmas resursi glabājas droši mākoņā. Backend: Supabase API — gatavs risinājums datu pārvaldībai un autorizācijai, savs serveris nav nepieciešams. Mindmap: jsMind (vai React Flow) — vizuālās mērķu kartes veidošanai. Iedvesma: Unsplash API (bildes). Progresu vizualizācija: Chart.js — grafiki un izpildes indikatoru vizualizācija.
